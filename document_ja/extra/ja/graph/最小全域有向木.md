# Minimum Arborescence（Directed MST／最小全域有向木）

指定した根から全頂点へ到達する、重み最小の全域有向木を求めます。

`Minimum Arborescence`または`Directed MST`と呼ばれます。このページは既存の日本語API`最小全域有向木`を説明します。

## import

~~~nim
import std/options
import atcoder/extra/ja/graph/最小全域有向木
~~~

## 使用例

~~~nim
let result = 最小全域有向木(
  edges,
  vertexCount,
  root,
)

if result.isSome:
  let (重み合計, 選択結果) = result.get
  echo 重み合計
  echo 選択結果
~~~

存在しない場合は`none`を返します。頂点番号と根は0-indexedです。
