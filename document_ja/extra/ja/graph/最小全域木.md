# Minimum Spanning Tree（最小全域木）

重み付き無向グラフの最小全域木をKruskal法で求めます。

このページは`atcoder/extra/graph/kruskal`の日本語facadeを説明します。既存の型と英語APIも再公開します。

## import

~~~nim
import atcoder/extra/ja/graph/最小全域木
~~~

## 使用例

~~~nim
let (重み合計, 選ばれた辺) = 最小全域木(g)

echo 重み合計
echo 選ばれた辺
~~~

返り値は最小全域木の重み合計と、選択された辺の列です。

辺数を \(E\) とすると計算量は

\[
O(E\log E)
\]

です。非連結グラフでは最小全域森を返します。
