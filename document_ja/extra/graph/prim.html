基本例

prim(g, s) は始点 s から Prim 法で最小全域木を求め、重み合計と選ばれた辺の列を返します。

import atcoder/extra/graph/graph_template
import atcoder/extra/graph/prim

var g = initGraph(4)
g.addBiEdge(0, 1, 1)
g.addBiEdge(1, 2, 2)
g.addBiEdge(0, 2, 5)
g.addBiEdge(2, 3, 1)

let (cost, es) = prim(g, 0)

doAssert cost == 4
doAssert es.len == 3
