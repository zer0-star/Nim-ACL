# AC(AtCoder) Library Document\n\n## インストール方法\n\n- zipファイルを解凍すると、`ac-library`フォルダ, そしてその中に`atcoder`フォルダが入っているはずです。\n- g++を使っている場合, `atcoder`フォルダを`main.nim`と同じ場所に置いて、`nim cpp main.nim --include:.`でコンパイルできます。  \n- 詳しくは [Appendix](./appendix.html) を参照してください。\n\n## お約束\n\n- 制約外の入力を入れたときの挙動はすべて未定義です。\n- このドキュメントでは長い型を便宜上短く書きます\n  - `unsigned int` → `uint`\n  - `long long` → `ll`\n  - `unsigned long long` → `ull`\n- $0^0$ は $1$ です\n- 明記されていない場合、多重辺や自己ループも入力可能です。\n\n## リスト\n\n`import atcoder/all` : 一括include\n\n### データ構造\n\n- [`import atcoder/fenwicktree`](./fenwicktree.html)\n-
