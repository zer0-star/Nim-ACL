# AtCoder Algorithm Lectures coverage

Checked: 2026-07-11

This page maps the official AtCoder Algorithm Lectures catalog to Nim-ACL implementations and documentation.

It does not reproduce lecture text, figures, or source code. Only titles, difficulty metadata, links, and original Nim-ACL status assessments are included.

- [Official AtCoder Algorithm Lectures index](https://info.atcoder.jp/entry/algorithm_lectures/index)

The detailed coverage table is maintained in the [Japanese page](../../document_ja/guide/atcoder_algorithm_lectures_coverage.html).

## Evaluation policy

- Official difficulty is treated as reference metadata, not an AtCoder color.
- Nim-ACL implementation priority also considers contest frequency, Library Checker coverage, maintenance cost, and API quality.
- External lecture contents are linked rather than copied.
