# Divide-and-Conquer Optimization

This is a functional-name forwarding import for `atcoder/extra/dp/monotone_minima`.

~~~nim
import atcoder/extra/dp/divide_and_conquer_optimization
~~~

It re-exports the following APIs:

- `monotoneMinima`
- `divideAndConquerOptimization`
- `divideAndConquerDpLayer`

See [Monotone Minima](monotone_minima.md) for the full documentation.
